# Solax/FoxESS V1 Hybrid/Fit/AC Can Bus Protocol

Can bus @ 500k - all Extended ID, little endian

## Frames

### Received frame 

> 0x1871 [0x01,  0x00,  0x01,  0x00,  0x00,  0x00,  0x00,  0x00]
